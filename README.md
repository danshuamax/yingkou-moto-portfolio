## 如何更新作品（推荐：一键同步）

### 最简单的方法

1. **把文件丢进文件夹**（见下表）  
2. **双击** `更新作品.command`  
3. 浏览器 **强刷**（`Cmd + Shift + R`）

也可以在终端执行：

```bash
cd ~/Downloads/营口机车影像
python3 sync-works.py
```

脚本会自动：
- **不规范文件名自动重命名**（DSC / ChatGPT / grok-video / 纯数字等 → `项目名-01.jpg`）  
- 压缩过大的写真图  
- **视频缺封面时自动截帧生成** `cover.jpg`  
- 扫描项目并重写 `works.js`  

重命名记录见 `rename-log.json`。也可手动放 `cover.jpg` 指定封面。

### 文件丢哪里

#### 精选作品三栏（摄影约拍 · 按子项目上传）

| 精选卡片 | 上传路径 | 内容 |
|----------|----------|------|
| **城市追光** | `photo/城市追光/项目名/*.mp4` | 动态跟拍视频 |
| **夜色锋芒** | `photo/夜色锋芒/项目名/*.jpg` | 夜晚拍摄 |
| **港湾日光** | `photo/港湾日光/项目名/*.jpg` | 白天拍摄 |

示例：

```
photo/港湾日光/芒果/DSC001.jpg
photo/港湾日光/胖胖/DSC08835.jpg
photo/夜色锋芒/夜色约拍/*.JPG
photo/城市追光/沿海公路/成片.mp4
```

点精选大卡 → **先出现项目卡片列表**（与 AIGC 相同）→ 再点项目看作品。

#### AIGC 按项目

```
assets/portfolio/aigc/video/项目名称/成片.mp4
assets/portfolio/aigc/design/项目名称/图1.jpg
```

### 改中文标题

编辑 `titles.json`（项目文件夹名或文件名 → 标题），再运行一次同步。
